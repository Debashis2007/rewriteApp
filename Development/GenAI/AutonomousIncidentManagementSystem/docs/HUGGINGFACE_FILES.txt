DEPLOYMENT OPTIONS

Option 1: Frontend-Only (RECOMMENDED) ⚡
───────────────────────────────────────

  3. Deploy

──────────────────────────────────

Time: 5-15 minutes
Size: ~1GB
Resources: High (2GB+ RAM)

  2. Push all directories to HF Space
  3. Deploy

================================================================================
QUICK START COMMANDS
================================================================================

Automated Deployment (Recommended):
  cd /Users/deb/Development/GenAI/AutonomousIncidentManagementSystem
  ./deploy-huggingface.sh

Manual Frontend-Only Deployment:
  git clone https://huggingface.co/spaces/<username>/autonomous-incident-management
  cd autonomous-incident-management
  cp -r ../AutonomousIncidentManagementSystem/frontend .
  cp ../AutonomousIncidentManagementSystem/Dockerfile.frontend-hf Dockerfile
  git add . && git commit -m "Deploy AIMS" && git push

Test Locally (Frontend):
  docker build -f Dockerfile.frontend-hf -t aims-frontend:latest .
  docker run -p 7860:7860 aims-frontend:latest

Access After Deployment:
  https://huggingface.co/spaces/<username>/autonomous-incident-management

================================================================================
TROUBLESHOOTING
================================================================================

Build Fails:
  1. Check Dockerfile syntax
  2. Verify all files exist
  3. Review Space Logs tab
  4. Check file sizes

App Crashes:
  1. Check Space logs
  2. Verify port 7860 is used
  3. Check environment variables
  4. Try redeploying

Can't Connect to API:
  1. Verify backend is running
  2. Check API endpoint URL
  3. Check CORS settings
  4. Verify network connectivity

Database Connection Issues:
  1. Check DATABASE_URL
  2. Verify database is running
  3. Check network access
